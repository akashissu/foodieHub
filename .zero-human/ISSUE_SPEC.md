# PAP-466: Develop a modern full-stack food delivery web application similar to Zomato.

Create a full-stack Zomato-style food delivery web app called FoodieHub using React/Next.js, TypeScript, Tailwind CSS, Framer Motion, Node.js, Express.js, MongoDB/PostgreSQL, JWT authentication, and REST APIs.

Include Home, Login, Signup, Restaurant Listing, Restaurant Details, Cart, Checkout, Order Success, Order Tracking, User Profile, Admin Dashboard, and Restaurant Owner Dashboard.

Features should include restaurant search, cuisine filters, ratings, menu items, add to cart, quantity update, coupon apply, checkout, order placement, order tracking, order history, admin restaurant management, food item management, user management, and restaurant owner order management.

Design should be modern, responsive, premium, clean, food-focused, and inspired by Zomato without directly copying it. Use red/coral accents, white background, rounded cards, soft shadows, attractive food images, smooth animations, loading states, hover effects, toast notifications, and mobile-first UI.
